checkInPlaceID:text:latitude:longitude:friendsOnly:

Отмечает пользователя в указанном месте

- (id)checkInPlaceID:(NSUInteger)placeID text:(NSString *)text latitude:(CGFloat)latitude longitude:(CGFloat)longitude friendsOnly:(NSUInteger)friendsOnly

Parameters

placeID

идентификатор места

text

комментарий к отметке длиной до 255 символов (переводы строк не поддерживаются)

latitude

географическая широта отметки, заданная в градусах (от -90 до 90)

longitude

географическая долгота отметки, заданная в градусах (от -180 до 180)

friendsOnly

1 — отметка будет доступна только друзьям, 0 — отметка будет доступна всем пользователям

Return Value

ответ на запрос в виде Foundation объекта
